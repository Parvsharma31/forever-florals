Add your photos here and they'll automatically be used by the site.

File naming suggestions (use these exact names for automatic wiring):

- Cuisines:
  - `it.jpg` (Italian)
  - `jp.jpg` (Japanese)
  - `mx.jpg` (Mexican)
  - `us.jpg` (American)

- Chefs:
  - `chef-aiko.jpg`
  - `chef-marco.jpg`
  - `chef-lena.jpg`

Notes:
- Recommended size: ~1200px wide for hero-like images or ~800px for regular cards.
- Use `.jpg` or `.png`. Keep filenames lowercase and replace spaces with hyphens.
- If a photo is missing, the site will gracefully fall back to the existing SVG illustrations.
- After adding images, refresh the page to see them appear.
